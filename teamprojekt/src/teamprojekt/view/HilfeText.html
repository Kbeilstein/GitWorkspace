Das 'Auswahl'-Feld

 

In der Combo-Box 'Sondierverfahren wählen' können Sie eins der vorgeschlagenen Verfahren

auswählen, dass in einer Animation gestartet werden kann.

Zur Auswahl stehen:

    - das Lineare Sondieren,

    - das Verallgemeinertes Lineare Sondieren,

    - das Quadratische Sondieren, 

    - ein alternierendes Quadratisches Sondieren und

    - das Doppel-Hashing.

 

Alle Verfahren unterscheiden sich, vor allem in der Kollessionsbehandlung und dementsprechend auch

in der Animation deutlich.

 

Die 'Arraygröße' bzw. 'Arraylänge' können Sie mit dem Spinner bestimmen. Sie können entweder

mit dem 'Minus' oder 'Plus' Button die Länge immer um ein Feld verringern oder erhöhen, oder Sie

geben in das Nummernfeld die gewünschte Größe direkt ein. Um das Array überschaubar zu halten,

akzeptiert das Nummernfeld des Spinners eine minimale Größe von 5 bis zur maximalen Größe von 19

des Arrays. Andere Größen werden nicht akzeptiert. (Standard: Arraygröße 11)

 

Eine geeignete Konstante 'c'  können Sie nur bei dem Verallgemeinerten Linearen Sondieren wählen,

ansonsten ist dieser Bereich inaktiv.

Das zu wählende 'c' hängt von der Arraylänge ab, welches Sie vorher wählen.

Die vorgeschlagen Konstanten, haben keinen gemeinsamen Teiler mit der Arraygröße um Schleifen zu

vermeiden.

 

Ist das Verfahren und die Größe des Arrays gewählt, können Sie mit dem 'Start' Button das Aktionsfeld

aktivieren. Der Start-Button und alle anderen Elemente in dem Auswahl-Feld sind zu jederzeit aktiv,

falls sie Änderungen, Anpassungen oder gar ein anderes Sondierverfahren auswählen und starten möchten.

 

Der 'Pseudo-Code' Button öffnet ein gesondertes Fenster, welches das ausgewählte Verfahren in

einem Java-Pseudo-Code wiederspiegelt.

Das Pseudo-Code Fenster enthält lediglich drei Methoden des jeweiligen Verfahrens. Diese wiederum

sind Zeilenweise Kommentiert, um die Arbeitsweise des Verfahrens bestmöglich verfolgen zu können.

Zusätzlich finden Sie eine Info mit kleinen Besonderheiten zum jeweiligen verfahren.

 

Mit dem 'Log laden' Button können Sie frühere abgespeicherten Logs wieder herstellen und

animieren lassen. Wenn Sie eine Log-Datei laden, ist es dennoch wichtig das Verfahren und die

Arraygröße zu wählen. Die Log-Datei kann in jedem beliebigen Sondierverfahren gewählt werden,

da Sie nicht auf ein Bestimmtes vorher gewähltes Verfahren zugeschnitten wird.

 

Das 'Actions'-Feld

 

Im Grunde lässt Sich dieses Feld in 3 Bereiche einteilen.

1) In dem oberen Bereich finden Sie das 'Animations' Feld, welches das Arrayfeld mit der gewählten

    Größe darstellt und in der alle Animationen stattfinden.

 

2) Im mittleren Bereich befindet sich das 'Kontroll' Feld. Hier können Sie den Ablauf mit verschieden

    Buttons steuern.

Der 'Zurück' Button, lässt die Animation Schritt für Schritt rückwärts laufen

    und stellt die vorherigen schritte wieder her.

Der 'Play/Stop' Button lässt die Animation stoppen und wieder weiterlaufen. (Standard: 'Play' Zustand)

    Hinweis ! Sobald eine Zahl hinzugefügt wurde startet die Animation und läuft schrittweise automatisch ab.

Der 'Vorwärts' Button lässt die Animation nach Schritt für Schritt vorwärtslaufen. Dieser ist erst aktiv

    wenn zuvor einmal der Stop-Button gedrückt wurde.

Der 'Geschwindigkeitsregler' lässt sich auch während der Animation bedienen um eine gewünschte

   Animationsgeschwindigkeit zu erzeugen.

Das 'Eingabe' Feld akzeptiert nur Zahlen zwischen 1 und 99, welche in der angestrebten Animation

    dargestellt werden sollen.

Der 'hinzufügen' Button leitet die vorhergewählte Eingabe in das Array und lässt die Animation

    beginnen.

Der 'suchen' Button lässt die vorhergewählte Eingabe im Array suchen.

Der 'löschen' Button lässt die vorhergewählte Eingabe im Array löschen.

 

3) Im unteren Bereich finden Sie das 'Log' Feld. Dieser Kommentiert jeden einzelnen Schritt der

    Animation und liefert eine Erklärung wie z.B. die Eingabe auf ein bestimmtes Feld gelangt.

 

Mit dem 'Log speichern' Button können Sie, die zuvor erstellte Animationsfolge in einem Log speichern.

 

Die Legende

 

Die Legende dient als Hilfestellung der zu animierenden Elemente.

    'Grün'    das Einfügen der Eingabe im Array ist möglich.

    'Rot'        das Einfügen der Eingabe ist nicht möglich, es tritt eine Kollision auf.

    'Blau'      die Eingabe wurde gefunden.

    'Orange' die zu suchende Eingabe befindet sich nicht in jeweiligen Feld.

    'Gelb'     die Eingabe wurde gelöscht und das Feld 'gelb' markiert
